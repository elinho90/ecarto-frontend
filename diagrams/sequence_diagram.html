Processus Clés de la Plateforme

1. Authentification JWT

sequenceDiagram actor U as 👤 Utilisateur participant F as 🌐 Angular Frontend participant AC as 🔑 AuthController participant SS as 🛡️ Spring Security participant DB as 🗄️ Base de Données U->>F: Saisir email + mot de passe F->>AC: POST /api/auth/login {email, password} AC->>SS: authenticate(credentials) SS->>DB: SELECT * FROM utilisateur WHERE email = ? DB-->>SS: Utilisateur trouvé (rôle, hash) SS->>SS: Vérifier BCrypt password SS-->>AC: ✅ Authentication validée AC->>AC: Générer JWT Token (expiration 24h) AC-->>F: 200 OK {token, refreshToken, utilisateur, rôle} F->>F: Stocker token dans localStorage F->>F: Configurer HttpInterceptor (Authorization: Bearer) F-->>U: ✅ Redirection vers Dashboard

2. Création et Visualisation d'un Site sur la Carte

sequenceDiagram actor CP as 👷 Chef de Projet participant F as 🌐 Angular Frontend participant SC as 📍 SiteController participant SV as ⚙️ SiteService participant DB as 🗄️ Base de Données CP->>F: Remplir formulaire (nom, adresse, lat, lng, statut) F->>F: Validation Angular Reactive Forms F->>SC: POST /api/sites {siteDto} Note over SC: @PreAuthorize("ADMIN, CHEF_DE_PROJET") SC->>SV: createSite(siteDto) SV->>SV: Valider coordonnées GPS SV->>DB: INSERT INTO site (nom, latitude, longitude, statut, ...) DB-->>SV: Site créé (id = 15) SV-->>SC: SiteDto response SC-->>F: 201 Created {site} F->>F: Ajouter marqueur Leaflet avec couleur statut Note over F: Actif=🟢 Inactif=🔴 Maintenance=🔵 Construction=🟠 F-->>CP: ✅ Site affiché sur la carte avec animation pulse

3. Génération d'un Rapport Projet

sequenceDiagram actor D as 🏛️ Décideur participant F as 🌐 Angular Frontend participant RC as 📈 RapportController participant RS as ⚙️ RapportService participant PS as 📁 ProjetService participant DB as 🗄️ Base de Données D->>F: Sélectionner un projet et demander un rapport F->>RC: POST /api/rapports {projetId, type} RC->>RS: generateRapport(projetId) RS->>PS: getProjetById(projetId) PS->>DB: SELECT * FROM projet WHERE id = ? DB-->>PS: Projet data RS->>DB: SELECT * FROM site WHERE projet_id = ? DB-->>RS: Liste des sites RS->>DB: SELECT * FROM task WHERE projet_id = ? DB-->>RS: Liste des tâches RS->>RS: Calculer statistiques (budget, avancement, KPIs) RS->>DB: INSERT INTO rapport (...) DB-->>RS: Rapport créé RS-->>RC: RapportDto RC-->>F: 200 OK {rapport} F-->>D: ✅ Affichage du rapport détaillé
